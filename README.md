# MONRU (Monitoring Ruangan)

## Deskripsi
**MONRU (Monitoring Ruangan)** adalah aplikasi untuk mencatat dan memantau hasil pengukuran kondisi ruangan secara terpusat.  
Aplikasi ini menyediakan formulir pengukuran untuk input parameter utama seperti **suhu**, **kelembapan relatif (RH)**, dan **diferensial tekanan (DP)**, serta menampilkan **dashboard monitoring** untuk melihat rekap dan status pelaksanaan pengukuran per ruangan pada periode tertentu, sehingga pengguna dapat memastikan pengukuran dilakukan sesuai kebutuhan dan mudah ditelusuri kembali.
